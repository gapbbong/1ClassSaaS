# 🚀 Supabase 마이그레이션 실행 가이드

계란 사 오시느라 고생 많으셨습니다! 선생님께서 외출하신 동안 제가 인프라 구축과 자동 이관 준비를 모두 마쳤습니다. 이제 아래 단계에 따라 마이그레이션을 시작해 볼까요?

## 1단계: 보안 키 입력 및 확인
제가 선생님이 주신 정보를 바탕으로 `.env` 파일을 이미 생성해 두었습니다. 혹시라도 수정이 필요하면 아래 파일을 확인해 주세요.
- **파일 경로:** `d:/App/OneClass25/.env`

## 2단계: 로컬 사진 폴더 준비
선생님께서 이미 사진들을 아래 폴더에 **'학번.jpg'** 형식으로 잘 넣어두셨으므로, 이 단계는 완료되었습니다!
- **폴더 위치:** `d:/App/OneClass25/25 학생 사진`

## 3단계: 데이터베이스 테이블 생성
Supabase 대시보드의 **SQL Editor**에 접속하여, 제가 작성한 `supabase_schema.sql`의 내용을 복사해서 실행(Run)해 주세요.
- **파일 경로:** `d:/App/OneClass25/supabase_schema.sql`

## 4단계: 사진 및 데이터 이관 실행
모든 준비가 끝났다면 터미널(CMD/PowerShell)에서 아래 명령어를 입력해 주세요.
```powershell
node migrate_photos.js
```

---
### 💡 참고 사항
- **이미지 최적화:** 모든 사진은 업로드 시 자동으로 가로 1,200px로 리사이징되어 용량을 아낍니다.
- **보안:** 모든 데이터는 RLS 정책에 의해 담임 선생님 본인만 접근할 수 있도록 보호됩니다.
- **문제 발생 시:** 제가 실시간으로 로그를 모니터링하고 있으니, 에러 메시지가 뜨면 바로 알려주세요!
