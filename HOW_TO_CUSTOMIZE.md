# OneClass 타 학교 도입 및 커스터마이징 가이드 🚀

`OneClass`는 소스 코드 수정 없이 **구글 시트의 설정만으로** 어느 학교에서나 즉시 사용 가능합니다.

## 1. 구글 시트 기본 구성 (Settings)

`Settings` 시트 탭에서 다음과 같이 데이터를 구성해 주세요.

| A열 (잘한 일) | B열 (못한 일) | D열 (설정키) | E열 (설정값) |
| :--- | :--- | :--- | :--- |
| 수업 태도가 좋음 | 지각 | `schoolName` | **[학교명 입력]** (예: 원클고등학교) |
| 친구를 도와줌 | 복장 불량 | `majorMapping` | **[아래 JSON 형식 참조]** |

### 학과 매핑 (majorMapping) 작성법
E열에 아래 형식을 복사하여 반별 학과명을 지정하세요.
```json
{
  "1": "IoT전기과", "2": "IoT전기과", "3": "IoT전기과",
  "4": "게임콘텐츠과", "5": "게임콘텐츠과", "6": "게임콘텐츠과"
}
```

## 2. 교사 정보 구성 (Teachers)

`Teachers` 라는 이름의 새 시트 탭을 만들고 아래 헤더에 맞춰 담임/부담임 정보를 입력하세요.

| 학년 | 반 | 담임 | 담임연락처 | 부담임 | 부담임연락처 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 1 | 1 | 홍길동 | 010-1234-5678 | 이순신 | 010-9876-5432 |
| ... | ... | ... | ... | ... | ... |

## 3. 구축 및 배포 순서

1.  **시트 복사**: 템플릿 구글 시트를 복사합니다.
2.  **GAS 배포**: 시트 내 [확장 프로그램] > [Apps Script]로 들어가 제공된 코드를 붙여넣고 [웹 앱으로 배포]합니다.
3.  **URL 연결**: 배포된 URL을 `src/js/config.js`의 `SCRIPT_URL`에 입력합니다.
4.  **Netlify 배포**: GitHub 저장소와 Netlify를 연결하여 배포하면 끝!
