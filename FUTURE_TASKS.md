# 🚀 추후 추가 작업 예정 사항

## 1. 생활기록 기능 강화
*   **데이터 로딩 안내**: 생활기록 페이지 진입 시 "잘한 일, 못한 일 항목을 불러오고 있습니다." 문구와 함께 부드러운 애니메이션/스피너 표시.
*   **기록 교사 자동화**: 
    *   현재의 '기록 교사(선택)' 입력 필드 제거.
    *   로그인한 사용자의 이메일 주소 중 `@` 앞부분(ID)을 자동으로 추출하여 기록자로 저장.
*   **반성문 사진 첨부 기능**:
    *   '반성문 이미지' 항목 추가 및 옆에 카메라 아이콘 배치.
    *   아이콘 클릭 시 카메라 촬영 또는 앨범 선택 가능하도록 구현.
    *   사진 선택 시 화면에 작은 썸네일 미리보기 표시.
    *   **이미지 최적화**: 서버 저장 공간 절약을 위해 가독성이 확보되는 최소한의 해상도와 크기로 자동 리사이징/압축 후 저장.
*   **기록 내역 조회**: 
    *   기록 목록에서 반성문 사진이 포함된 항목은 '반성문 보기' 버튼 표시 및 클릭 시 이미지 확대 보기 기능.

## 2. 권한 및 보안 고도화 (Role-Based Access Control)
사용자별 담당 직무에 따른 데이터 접근 제어:
*   **김변정 선생님 (상담)**: 모든 학생의 데이터 중 '알레르기 정보'만 열람 가능.
*   **배움터지킴이 선생님 (보안)**: 학생들의 '근태 기록(조퇴/외출 등)'만 열람 가능.
*   **관리자 (이경원, 류기현)**: 시스템 내 모든 학생의 모든 데이터 및 설정에 대한 전체 권한.
*   **담임 교사**: 본인 반 학생들의 모든 정보 열람 및 수정 가능. 단, 타 학급 학생의 상세 정보는 접근 불가.
*   **생활지도부장 (이상수 선생님)**: 관리자와 동일하게 모든 학생 데이터 및 항목 열람 권한 부여.
